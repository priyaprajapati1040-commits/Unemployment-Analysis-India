Project: Unemployment Analysis in India

Objective:
Analyze unemployment trends in India using Python.

Tools Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

Analysis Performed:
- Data Cleaning
- Unemployment Distribution
- State-wise Analysis
- Urban vs Rural Comparison
- Monthly Trends
- COVID-19 Impact Analysis
- Correlation Heatmap

Conclusion:
COVID-19 significantly increased unemployment rates in India. Urban areas were more affected than rural areas.
